# Metal Vida - Sistema Interno de Orcamento

Aplicacao estatica para calculo gerencial de orcamentos internos.

## Arquivos principais

- index.html: tela do sistema
- app.css: estilos do sistema
- app.js: regras de calculo e interacoes

## Fluxo de uso

1. Preencher dados do cliente e do equipamento.
2. Ajustar parametros financeiros.
3. Conferir custos de materiais e servicos.
4. Adicionar item ao orcamento.
5. Exportar CSV ou imprimir em PDF.

## Observacao

Antes de enviar proposta formal, validar escopo tecnico, frete, impostos, prazo e condicoes comerciais.
